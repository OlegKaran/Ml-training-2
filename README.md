# Ml-training-2
Участие в kaggle-соревновании с целью попасть в верхнюю четверть таблицы (25%)

Название соревнования: Spaceship Titanic https://www.kaggle.com/c/spaceship-titanic

Описание подхода:
Я начал с логистической регрессии, затем я обучил другие модели: KNN,SVM,подобрав гиперпараметры с помощью GridSearchCV. 

Затем я обучил RandomForest: сначала без GridSearchCV, потом с ним. Подбор гиперпараметров не помог и Accuracy модели уменьшилась с 0.7995 до 0.7952.

По такой же схеме я обучил XGBoost: подбор гиперпараметров также не помог: Accuracy уменьшилась с 0.8090 до 0.7952.

После этого я обучил CatBoost без подбора гиперпараметров и с ним: GridSearchCV сработал и показал лучший результат среди всех моделей

Метамодель линейная регрессия не улучшила результат.

Работа с признаками:
Создание колонки с суммарными тратами 'Totalspend' улучшило модели, заполнение колонок с тратами: 'RoomService','FoodCourt','ShoppingMall','Spa','VRDeck', '0',если в колонке 'пусто', оказалось лучше заполнения этих же колонок медианами.
